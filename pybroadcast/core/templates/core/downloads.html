Os executáveis abaixo não são instaladores. São programas autocontidos que são carregados em memória.

pybroadcast_client.exe (Versão de produção / background)

Esta versão, ao ser executada, é carregada na memória RAM, sendo executada em background.

Esta versão deverá ser executada, via CMD, para debug de eventuais problemas.
